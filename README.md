# configure the database for testing?
 DB url is already set for local machine, make sure mongodb service is running on your local machine
## For this you will need 2 terminal windows.

	a. First terminal:
		1. CD into /rentalbot
		2. npm install
		3. node server.js

	b. Second terminal:
		1. CD into /rentalbot/client
		2. npm install
		3. npm start
